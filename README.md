# Мета-обучение для выбора алгоритмов классификации

## Описание
Проект реализует систему мета-обучения для автоматического выбора наилучшего алгоритма классификации.

## Структура
- `src/` - исходный код
- `notebooks/` - Jupyter ноутбуки
- `data/` - данные
- `results/` - результаты
- `config/` - конфигурации
